# Web Entry Form Assignment

## Objective

Create a **Web Entry Form** for any entity of your choice to gather information from users.

You must design the form using **HTML form elements** and ensure that all major input types and attributes are demonstrated.

Use an **HTML Table** to properly align labels and input controls.

---

# Suggested Form Topics

Choose **one** of the following entities for your form:

1. Customer KYC  
2. Loan Application  
3. Bank Account Opening  
4. Passport Application  
5. Driving License Application  
6. College Admission Form  

---

# Functional Requirements

Your form must include the following:

### 1. Form Inputs

Use as many input types as possible, including:

- Text
- Password
- Email
- Number
- Date
- Radio buttons
- Checkboxes
- File upload
- Dropdown (select)
- Textarea
- Submit button
- Reset button

---

### 2. Input Attributes

Use appropriate HTML attributes such as:

- `required`
- `placeholder`
- `maxlength`
- `min`
- `max`
- `pattern`
- `readonly`
- `disabled`
- `autocomplete`
- `checked`

---

### 3. Layout Requirement

Use an **HTML Table** to align:

- Labels
- Input fields
- Form controls

Example structure:

```
Label | Input Control
```

Each row should contain a label and its corresponding form control.

---

# Example Information Fields

Depending on the form type, collect details such as:

### Personal Details

- Full Name  
- Date of Birth  
- Gender  
- Email  
- Phone Number  

### Address Information

- Address Line  
- City  
- State  
- Postal Code  
- Country  

### Identity Details

- Government ID Number  
- Upload Supporting Documents  

### Additional Information

- Employment Status  
- Income Details  
- Course Selection / Loan Amount / License Category (based on form type)

---

# Validation Requirements

The form must include:

- Mandatory field validation
- Proper input types
- Pattern validation where applicable
- Numeric limits where required

---

# Output Expectations

The final form should:

- Be cleanly aligned using a table
- Contain multiple input types
- Demonstrate correct use of HTML attributes
- Collect complete information for the selected entity
